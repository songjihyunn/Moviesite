## 프로젝트 소개
- 영화에 대한 자유로운 이야기를 나눌 수 있는 자유 게시판
- 영화에 대한 정보가 담겨 있는 영화 게시판
- 여러 공지사항을 입력 할 수 있는 공지사항 게시판
- 궁금한 것이 있을 때 사용 가능한 Q&A 게시판

## 일정관리
- 1일차: 주제 선정 및 개발 일정 수립
- 2일차: 개발환경 설정, DB설계
- 3일차: 페이지 구상
- 4 ~ 5일차
  - 회원가입 페이지
  - 메인 페이지
- 6 ~ 8일차
    - 자유게시판
    - 공지사항
    - Q&A
- 9 ~ 10일차
    - 영화 상세 페이지
    - 오류 수정
- 11 ~ 12일차
    - 전체 검색 페이지
    - 아이디/비밀번호 찾기 페이지
- 13 ~ 14일차
    - 메인페이지 수정
